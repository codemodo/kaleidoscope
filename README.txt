============
Kaleidoscope
============

--------
Overview
--------

A wondrous, mind bending eye-gasm!

... okay, maybe a bit of an exaggeration.

To get some experience with Swing and the MVC model, we created a simple, randomly-generated, interactive kaleidoscope.  We had some fun with it by including our TAs' and professor's faces.
